## Object Oriented Programming

객체들 끼리 커뮤니케이션하도록 프로그래밍하는 것.

1. 클래스 단위로 개발하며 재사용성이 높아진다.
2. 확장성과 유지보수성이 좋다.
3. 데이터를 가진 변수와 변수를 수정하는 함수로 클래스가 구성된다.
4. 클래스는 데이터와 함수를 묶어놓은 템플릿이라 할 수 있다.
